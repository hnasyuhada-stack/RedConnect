# RedConnect – Blood Donation Management System  
A Java-based web application built using **JSP, Servlets, MVC, and MySQL**, developed as part of UiTM CSC584 – Enterprise Programming (Group Project).

RedConnect helps healthcare staff manage donors, track blood stock levels, and monitor blood usage through a centralized system.

---

## Features
- **User Registration & Login** (with validation)
- **Staff Authentication**
- **Dashboard with Analytics**
  - Bar Chart (Stock by blood type)
  - Pie Chart (Blood distribution)
- **Add New Donor**
- **View All Donor Details**
  - Search by Name
  - Search by Blood Group
  - Print Report
- **Increase / Decrease Blood Stock**
- **Stock Details Overview**
- **Secure Logout**
